Dicas para o Dia a Dia com Métodos JavaScript

Dicas para o Dia a Dia com Métodos JavaScript

Abstract

Esta nota reúne métodos e recursos de JavaScript que aparecem com frequência no dia a dia.
A proposta é ensinar do básico ao avançado com linguagem clara, direta e progressiva.
Cada método inclui explicação e 3 exemplos de uso.

Básico

Info

Nesta fase, a ideia é entender métodos que ajudam a lidar com textos, listas e verificações simples.

toUpperCase()

O que faz: transforma um texto em letras maiúsculas.

Como entender: imagine que você quer destacar uma palavra, como se estivesse escrevendo um aviso. Esse método faz esse destaque convertendo tudo para caixa alta.

Explicação: ele retorna uma nova string. O valor original continua igual, a menos que você salve o resultado em outra variável.

Exemplo 1

const nome = "ana";
console.log(nome.toUpperCase()); // ANA

Exemplo 2

const aviso = "promoção";
console.log(aviso.toUpperCase()); // PROMOÇÃO

Exemplo 3

const cidade = "Recife";
const destaque = cidade.toUpperCase();
console.log(destaque); // RECIFE

toLowerCase()

O que faz: transforma um texto em letras minúsculas.

Como entender: é útil quando textos chegam misturados, com maiúsculas e minúsculas, e você quer padronizar antes de comparar.

Explicação: também retorna uma nova string, sem mudar a original.

Exemplo 1

const texto = "JAVASCRIPT";
console.log(texto.toLowerCase()); // javascript

Exemplo 2

const email = "USUARIO@SITE.COM";
console.log(email.toLowerCase()); // usuario@site.com

Exemplo 3

const resposta = "Sim";
console.log(resposta.toLowerCase() === "sim"); // true

trim()

O que faz: remove espaços do começo e do fim de um texto.

Como entender: às vezes o valor parece certo, mas vem com espaços escondidos. Esse método limpa a borda do texto.

Explicação: ele não remove espaços do meio da frase. Apenas das extremidades.

Exemplo 1

const nome = "   Carlos   ";
console.log(nome.trim()); // Carlos

Exemplo 2

const senha = " 123456 ";
console.log(senha.trim()); // 123456

Exemplo 3

const entrada = "   ok";
console.log(entrada.trim() === "ok"); // true

includes()

O que faz: verifica se um texto ou array contém um valor.

Como entender: pense nele como uma pergunta direta: "isso está aqui dentro?".

Explicação: o retorno sempre será true ou false.

Exemplo 1

const frase = "Estou aprendendo JavaScript";
console.log(frase.includes("JavaScript")); // true

Exemplo 2

const frutas = ["maçã", "banana", "uva"];
console.log(frutas.includes("banana")); // true

Exemplo 3

const senha = "abc123";
console.log(senha.includes("@")); // false

slice()

O que faz: recorta uma parte de uma string ou array.

Como entender: é como pegar apenas um pedaço de algo maior, sem mexer no todo.

Explicação: você informa o ponto inicial e, opcionalmente, o final. O final não entra no resultado.

Exemplo 1

const palavra = "computador";
console.log(palavra.slice(0, 4)); // comp

Exemplo 2

const numeros = [10, 20, 30, 40, 50];
console.log(numeros.slice(1, 3)); // [20, 30]

Exemplo 3

const codigo = "ABC-1234";
console.log(codigo.slice(4)); // 1234

push()

O que faz: adiciona um item no final do array.

Como entender: imagine uma fila em que um novo elemento entra no fim.

Explicação: esse método altera o array original.

Exemplo 1

const lista = [1, 2, 3];
lista.push(4);
console.log(lista); // [1, 2, 3, 4]

Exemplo 2

const nomes = ["Ana", "Bruno"];
nomes.push("Carla");
console.log(nomes); // ["Ana", "Bruno", "Carla"]

Exemplo 3

const tarefas = [];
tarefas.push("estudar");
console.log(tarefas); // ["estudar"]

pop()

O que faz: remove o último item do array.

Como entender: se push() coloca no fim, pop() tira do fim.

Explicação: ele altera o array e ainda devolve o item removido.

Exemplo 1

const numeros = [1, 2, 3];
numeros.pop();
console.log(numeros); // [1, 2]

Exemplo 2

const frutas = ["maçã", "banana", "uva"];
const removida = frutas.pop();
console.log(removida); // uva

Exemplo 3

const historico = ["inicio", "meio", "fim"];
historico.pop();
console.log(historico); // ["inicio", "meio"]

indexOf()

O que faz: encontra a posição de um item.

Como entender: ajuda a responder "em que lugar isso está?".

Explicação: se não encontrar o valor, retorna -1.

Exemplo 1

const letras = ["a", "b", "c"];
console.log(letras.indexOf("b")); // 1

Exemplo 2

const palavra = "javascript";
console.log(palavra.indexOf("s")); // 4

Exemplo 3

const cores = ["azul", "verde"];
console.log(cores.indexOf("vermelho")); // -1

Intermediário

Tip

Aqui entram os métodos mais usados no trabalho com listas de dados, cadastros, filtros e transformações.

forEach()

O que faz: percorre todos os itens de um array.

Como entender: ele olha item por item, como se você passasse por cada elemento de uma lista manualmente.

Explicação: é bom para executar ações, mas não cria um novo array.

Exemplo 1

const nomes = ["Ana", "Bruno", "Carla"];

nomes.forEach((nome) => {
  console.log(nome);
});

Exemplo 2

const numeros = [1, 2, 3];

numeros.forEach((numero) => {
  console.log(numero * 2);
});

Exemplo 3

const compras = ["arroz", "feijão", "café"];

compras.forEach((item, indice) => {
  console.log(indice, item);
});

map()

O que faz: cria um novo array transformando cada elemento.

Como entender: você pega uma lista e devolve outra lista com os itens modificados.

Explicação: é um dos métodos mais importantes para trabalhar com dados sem alterar a estrutura original.

Exemplo 1

const numeros = [1, 2, 3];
const dobrados = numeros.map((n) => n * 2);

console.log(dobrados); // [2, 4, 6]

Exemplo 2

const nomes = ["ana", "bruno"];
const emMaiusculo = nomes.map((nome) => nome.toUpperCase());

console.log(emMaiusculo); // ["ANA", "BRUNO"]

Exemplo 3

const precos = [10, 20, 30];
const comDesconto = precos.map((preco) => preco * 0.9);

console.log(comDesconto); // [9, 18, 27]

filter()

O que faz: cria um novo array com itens que passam em uma condição.

Como entender: pense como uma peneira. O que atende à regra passa, o resto fica de fora.

Explicação: muito útil para buscas e seleções.

Exemplo 1

const idades = [12, 18, 25, 15];
const maiores = idades.filter((idade) => idade >= 18);

console.log(maiores); // [18, 25]

Exemplo 2

const produtos = ["mouse", "teclado", "monitor"];
const comM = produtos.filter((produto) => produto.includes("m"));

console.log(comM); // ["mouse", "monitor"]

Exemplo 3

const numeros = [1, 2, 3, 4, 5, 6];
const pares = numeros.filter((numero) => numero % 2 === 0);

console.log(pares); // [2, 4, 6]

find()

O que faz: encontra o primeiro item que atende a uma condição.

Como entender: ele procura até achar o primeiro resultado válido e então para.

Explicação: se nada for encontrado, o retorno é undefined.

Exemplo 1

const numeros = [4, 7, 10, 13];
const primeiroPar = numeros.find((numero) => numero % 2 === 0);

console.log(primeiroPar); // 4

Exemplo 2

const usuarios = [
  { nome: "Ana", ativo: false },
  { nome: "Bruno", ativo: true },
];

const usuarioAtivo = usuarios.find((usuario) => usuario.ativo);
console.log(usuarioAtivo); // { nome: "Bruno", ativo: true }

Exemplo 3

const frutas = ["maçã", "banana", "uva"];
const resultado = frutas.find((fruta) => fruta === "pera");

console.log(resultado); // undefined

some()

O que faz: verifica se pelo menos um item atende à condição.

Como entender: é como perguntar: "existe ao menos um caso assim?".

Explicação: retorna true ou false.

Exemplo 1

const numeros = [1, 3, 5, 8];
console.log(numeros.some((n) => n % 2 === 0)); // true

Exemplo 2

const usuarios = [
  { nome: "Ana", admin: false },
  { nome: "Bruno", admin: true },
];

console.log(usuarios.some((usuario) => usuario.admin)); // true

Exemplo 3

const notas = [4, 5, 6];
console.log(notas.some((nota) => nota >= 7)); // false

every()

O que faz: verifica se todos os itens atendem à condição.

Como entender: aqui a pergunta muda para: "todos passam nessa regra?".

Explicação: também retorna true ou false.

Exemplo 1

const notas = [8, 9, 10];
console.log(notas.every((nota) => nota >= 7)); // true

Exemplo 2

const palavras = ["sol", "mar", "ceu"];
console.log(palavras.every((palavra) => palavra.length === 3)); // true

Exemplo 3

const numeros = [2, 4, 5, 8];
console.log(numeros.every((numero) => numero % 2 === 0)); // false

reduce()

O que faz: transforma vários itens em um único resultado.

Como entender: imagine juntar várias pequenas partes para formar um total final.

Explicação: ele é poderoso porque pode somar, agrupar, contar ou montar novos objetos.

Exemplo 1

const valores = [10, 20, 30];
const soma = valores.reduce((acumulador, valor) => acumulador + valor, 0);

console.log(soma); // 60

Exemplo 2

const palavras = ["eu", "gosto", "de", "café"];
const frase = palavras.reduce((texto, palavra) => texto + " " + palavra);

console.log(frase); // eu gosto de café

Exemplo 3

const numeros = [1, 2, 3, 4];
const totalPares = numeros.reduce((acumulador, numero) => {
  return numero % 2 === 0 ? acumulador + 1 : acumulador;
}, 0);

console.log(totalPares); // 2

sort()

O que faz: ordena os itens de um array.

Como entender: é como organizar livros em uma prateleira ou números em ordem crescente.

Explicação: com números, o ideal é usar uma função de comparação para evitar resultados inesperados.

Exemplo 1

const letras = ["c", "a", "b"];
letras.sort();

console.log(letras); // ["a", "b", "c"]

Exemplo 2

const numeros = [30, 5, 100, 20];
numeros.sort((a, b) => a - b);

console.log(numeros); // [5, 20, 30, 100]

Exemplo 3

const nomes = ["Carlos", "Ana", "Bruno"];
nomes.sort();

console.log(nomes); // ["Ana", "Bruno", "Carlos"]

Object.keys()

O que faz: retorna as chaves de um objeto em forma de array.

Como entender: mostra os nomes dos campos do objeto.

Explicação: é útil quando você quer percorrer ou inspecionar a estrutura do objeto.

Exemplo 1

const usuario = { nome: "Ana", idade: 30 };
console.log(Object.keys(usuario)); // ["nome", "idade"]

Exemplo 2

const produto = { titulo: "Livro", preco: 50, estoque: 10 };
console.log(Object.keys(produto)); // ["titulo", "preco", "estoque"]

Exemplo 3

const configuracao = { tema: "escuro", idioma: "pt-BR" };
Object.keys(configuracao).forEach((chave) => {
  console.log(chave);
});

Object.values()

O que faz: retorna os valores de um objeto em forma de array.

Como entender: se Object.keys() mostra os nomes, esse mostra o conteúdo guardado nesses nomes.

Explicação: ajuda a trabalhar dados de objetos como listas.

Exemplo 1

const usuario = { nome: "Ana", idade: 30 };
console.log(Object.values(usuario)); // ["Ana", 30]

Exemplo 2

const notas = { prova1: 8, prova2: 9, prova3: 7 };
console.log(Object.values(notas)); // [8, 9, 7]

Exemplo 3

const temperaturas = { manha: 22, tarde: 29, noite: 24 };
const lista = Object.values(temperaturas);
console.log(lista); // [22, 29, 24]

Avançado

Warning

Aqui aparecem métodos e recursos muito úteis para código moderno, manipulação mais profunda e assincronismo.

flat()

O que faz: achata arrays que têm outros arrays dentro deles.

Como entender: em vez de uma lista com sublistas, você transforma tudo em uma lista mais simples.

Explicação: ajuda quando os dados chegam agrupados em vários níveis.

Exemplo 1

const lista = [1, 2, [3, 4]];
console.log(lista.flat()); // [1, 2, 3, 4]

Exemplo 2

const turmas = [["Ana", "Bruno"], ["Carla", "Diego"]];
console.log(turmas.flat()); // ["Ana", "Bruno", "Carla", "Diego"]

Exemplo 3

const itens = ["a", ["b", ["c"]]];
console.log(itens.flat(2)); // ["a", "b", "c"]

flatMap()

O que faz: transforma e achata ao mesmo tempo.

Como entender: primeiro você modifica cada item, depois junta tudo em uma estrutura só.

Explicação: evita fazer map() e flat() separadamente em muitos casos.

Exemplo 1

const frases = ["olá mundo", "bom dia"];
const palavras = frases.flatMap((frase) => frase.split(" "));

console.log(palavras); // ["olá", "mundo", "bom", "dia"]

Exemplo 2

const numeros = [1, 2, 3];
const pares = numeros.flatMap((numero) => [numero, numero * 2]);

console.log(pares); // [1, 2, 2, 4, 3, 6]

Exemplo 3

const categorias = ["frutas,verduras", "bebidas"];
const itens = categorias.flatMap((grupo) => grupo.split(","));

console.log(itens); // ["frutas", "verduras", "bebidas"]

Array.from()

O que faz: cria um array a partir de algo que se comporta como lista.

Como entender: pega uma estrutura parecida com coleção e transforma em array de verdade.

Explicação: isso permite usar métodos de array em valores que antes não tinham esses métodos.

Exemplo 1

const texto = "abc";
console.log(Array.from(texto)); // ["a", "b", "c"]

Exemplo 2

const numeros = Array.from([1, 2, 3], (n) => n * 10);
console.log(numeros); // [10, 20, 30]

Exemplo 3

const conjunto = new Set([1, 2, 2, 3]);
console.log(Array.from(conjunto)); // [1, 2, 3]

Set

O que faz: armazena valores únicos, sem repetição.

Como entender: pense em uma caixa onde o mesmo item só pode entrar uma vez.

Explicação: é muito útil para remover duplicados.

Exemplo 1

const numeros = new Set([1, 2, 2, 3]);
console.log(numeros); // Set(3) { 1, 2, 3 }

Exemplo 2

const nomes = ["Ana", "Ana", "Bruno"];
const unicos = [...new Set(nomes)];

console.log(unicos); // ["Ana", "Bruno"]

Exemplo 3

const tags = new Set();
tags.add("javascript");
tags.add("javascript");
tags.add("node");

console.log(tags); // Set(2) { "javascript", "node" }

Promise.then()

O que faz: executa algo depois que uma Promise termina com sucesso.

Como entender: é como marcar uma ação para acontecer depois que uma tarefa for concluída.

Explicação: muito usado para lidar com resultados assíncronos.

Exemplo 1

Promise.resolve("carregado").then((resultado) => {
  console.log(resultado); // carregado
});

Exemplo 2

const numero = Promise.resolve(10);

numero.then((valor) => {
  console.log(valor * 2); // 20
});

Exemplo 3

Promise.resolve("ana")
  .then((nome) => nome.toUpperCase())
  .then((nomeFormatado) => {
    console.log(nomeFormatado); // ANA
  });

Promise.catch()

O que faz: captura erros de uma Promise.

Como entender: se algo deu errado no caminho, esse método entra para tratar a falha.

Explicação: é essencial para evitar erros sem tratamento em fluxos assíncronos.

Exemplo 1

Promise.reject("falha ao carregar")
  .catch((erro) => {
    console.log(erro); // falha ao carregar
  });

Exemplo 2

Promise.resolve("ok")
  .then(() => {
    throw new Error("erro inesperado");
  })
  .catch((erro) => {
    console.log(erro.message); // erro inesperado
  });

Exemplo 3

const busca = Promise.reject("servidor indisponível");

busca.catch((erro) => {
  console.log("Aviso:", erro);
});

Promise.all()

O que faz: espera várias Promises terminarem.

Como entender: imagine várias tarefas acontecendo ao mesmo tempo, e você só continua depois que todas terminam.

Explicação: é ótimo quando as tarefas são independentes.

Exemplo 1

const p1 = Promise.resolve("A");
const p2 = Promise.resolve("B");

Promise.all([p1, p2]).then((resultado) => {
  console.log(resultado); // ["A", "B"]
});

Exemplo 2

const usuario = Promise.resolve({ nome: "Ana" });
const pedidos = Promise.resolve([1, 2, 3]);

Promise.all([usuario, pedidos]).then(([dadosUsuario, listaPedidos]) => {
  console.log(dadosUsuario, listaPedidos);
});

Exemplo 3

const nota1 = Promise.resolve(8);
const nota2 = Promise.resolve(9);
const nota3 = Promise.resolve(7);

Promise.all([nota1, nota2, nota3]).then((notas) => {
  console.log(notas); // [8, 9, 7]
});

async / await

O que faz: permite escrever código assíncrono de forma mais direta.

Como entender: em vez de encaixar várias funções .then(), você escreve um fluxo mais próximo de passos normais.

Explicação: melhora bastante a leitura do código.

Exemplo 1

async function carregarMensagem() {
  const mensagem = await Promise.resolve("dados prontos");
  console.log(mensagem);
}

carregarMensagem();

Exemplo 2

async function dobrarValor() {
  const valor = await Promise.resolve(5);
  console.log(valor * 2); // 10
}

dobrarValor();

Exemplo 3

async function mostrarNome() {
  const nome = await Promise.resolve("Ana");
  console.log(nome.toUpperCase()); // ANA
}

mostrarNome();

try...catch

O que faz: captura erros para o programa não quebrar sem controle.

Como entender: é como colocar uma proteção ao redor de um trecho de código que pode falhar.

Explicação: costuma aparecer muito junto com async/await.

Exemplo 1

try {
  throw new Error("algo deu errado");
} catch (erro) {
  console.log(erro.message); // algo deu errado
}

Exemplo 2

async function executar() {
  try {
    const resposta = await Promise.reject("falha");
    console.log(resposta);
  } catch (erro) {
    console.log("Erro:", erro);
  }
}

executar();

Exemplo 3

try {
  const usuario = JSON.parse("texto invalido");
  console.log(usuario);
} catch (erro) {
  console.log("JSON inválido");
}

Operador spread ...

O que faz: espalha valores de arrays, objetos ou argumentos.

Como entender: é uma forma prática de abrir uma estrutura e reaproveitar seu conteúdo.

Explicação: muito usado em cópias rasas, junções e atualização de dados.

Exemplo 1

const numeros = [1, 2, 3];
const copia = [...numeros, 4];

console.log(copia); // [1, 2, 3, 4]

Exemplo 2

const usuario = { nome: "Ana", idade: 30 };
const atualizado = { ...usuario, idade: 31 };

console.log(atualizado); // { nome: "Ana", idade: 31 }

Exemplo 3

const grupo1 = ["Ana", "Bruno"];
const grupo2 = ["Carla", "Diego"];
const todos = [...grupo1, ...grupo2];

console.log(todos); // ["Ana", "Bruno", "Carla", "Diego"]

Desestruturação

O que faz: extrai dados de arrays e objetos de forma simples.

Como entender: em vez de pegar item por item manualmente, você separa o que precisa de uma vez.

Explicação: não é um método, mas é um recurso muito presente no JavaScript moderno.

Exemplo 1

const usuario = { nome: "Ana", idade: 30 };
const { nome, idade } = usuario;

console.log(nome, idade); // Ana 30

Exemplo 2

const numeros = [10, 20, 30];
const [primeiro, segundo] = numeros;

console.log(primeiro, segundo); // 10 20

Exemplo 3

const produto = { titulo: "Livro", preco: 50 };
const { titulo } = produto;

console.log(titulo); // Livro

Como Estudar Esta Nota

Example

Uma boa ordem de estudo para o dia a dia:

  1. Comece pelos métodos de string.
  2. Depois avance para métodos de array.
  3. Em seguida pratique transformação de dados com map(), filter() e reduce().
  4. Depois trabalhe objetos com Object.keys() e Object.values().
  5. Por fim, entre em assincronismo com Promise, async/await e try...catch.

Dica de Didática

Success

Ao ensinar, use sempre esta sequência:
o que é -> para que serve -> como pensar -> exemplos reais.

  • Mostre primeiro o valor original.
  • Depois aplique o método.
  • Em seguida compare o antes e o depois.
  • Destaque se o método altera ou não o valor original.
  • Use exemplos comuns: nomes, listas, compras, notas, senhas, mensagens e produtos.
